5. 利用Hexo生成博客并部署到Github

① 安装node
② 安装Git
③ 打开http://npm.taobao.org
④ 复制使用说明npm install -g cnpm –registry=https://registry.npm.taobao.org
⑤ 在cmd中粘贴执行命令
⑥ 执行cnpm install –g hexo-cli
⑦ 执行cd /d \D:\嵌入式软件开发2017春\MD\hexo
⑧ 执行hexo generate
⑨ 执行hexo server
⑩ 执行hexo deploy
