git的使用

2017-03-28

一、实验目的

1.掌握git使用

2.掌握使用Github作为远程仓库

二、实验任务

练习本地git操作

实验步骤：

① 下载github客户端
② 在github中新建一个text项目
③ 把text克隆到桌面
④ 修改本地text内容
⑤ 在github客户端中添加修改信息并提交
⑥ 将修改同步提交到github上

用SSH同步本地与Github仓库

实验结果：

① 创建SSH Key
② 登录github，打开setting——SSH and GPG keys，再将id_rsa.pub文件的内容复制粘贴到SSH Key
③ 登录github，新建一个Repository名为git
④ 在本地的git仓库下执行git remote add ori git@github.com:ccxiaoqiang/git.git
⑤ 执行git push –u ori master,将本地git仓库的内容推送到github页面

四、实验小结

通过本次实验，我学会了通过github客户端在本地修改文件，并将修改后的文件同步到github页面，这样不仅能看到每一次文件修改的信息，也能随时找回被误删的文件。用SSH同步本地github仓库，能够将本地git中修改的内容同步到github页面。在实验期间，我遇到了很多问题，通过查看指令执行错误以及上网搜索错误解决方法，最终都被一一克服。
