使用Github管理Android项目

2017-04-11

一、实验目的

1. 掌握Android Studio中Git使用

2. 掌握Github管理项目

二、实验任务

1. 在Android Studio中新建项目

2. 添加VCS中的Git支持

3. 发布项目到Github

4. 在Android Studio中进行修改代码、提交、新建分支、合并等操作

5. 推送项目到Github

6. 如有多人合作，处理pull request

三、实验步骤与结果

  1. 在Android Studio新建项目，取名为Gourmet。
  2. 在Android Studio中的VCS中打开Import into Version Control—Share Project on Github，输入自己的github的帐号密码。
  3. 点击share，将本地Android Studio中的工程文件发布到github上。
  4. 对代码进行修改，修改在string.xml中修改全局变量，修改后字体成蓝色。
  5. 鼠标右击，选择Git—Commit File—commit and push。

    之后github进行同步修改，可以看到修改记录的信息和时间。

四、实验小结

通过本次实验，我学会了如何将本地的工程文件上传到github上。这给工程的修改带来了极大的方便，不再受地方的约束。只要有网络的地方就可以从github上下载代码到本地，再进行修改，修改完后再上传至github保存，还能显示每次的修改信息。
